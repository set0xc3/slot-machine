## Требования

- **Стандарт языка:** C++17.
- **Компилятор:** Clang.
- **Система сборки:** CMake.
- **Графический рендер:** OpenGL Core 4.6.

## Зависимости библиотек

- **SDL2** (2.32.2)
- **SDL2_image** (2.8.8)
- **GLAD** (0.1.36)

## Сборка

```bash
cmake -B build
cmake --build build --config Release
```
